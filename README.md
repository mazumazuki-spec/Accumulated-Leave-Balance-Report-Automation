# Accumulated Leave Balance Report Automation

ระบบ Automation สำหรับจัดทำรายงานวันหยุดสะสมพนักงานหลายสาขา
โดยใช้ Python ร่วมกับ Excel Automation เพื่อช่วยลดขั้นตอน Manual Report Preparation

---

## Project Overview

โปรเจกต์นี้จำลองกระบวนการจัดทำ Monthly Leave Balance Report
ตั้งแต่ Data Cleaning, Report Generation, Consolidation
จนถึง Final Report Publishing

Workflow:

Raw Data
↓
Data Cleaning
↓
Branch Report Generation
↓
Consolidated Summary
↓
Final Report Output


---

## Features

- Clean และ Transform ข้อมูลพนักงาน
- Generate Excel Report จาก Template
- Maintain Excel Formatting และ Formula
- Create Branch Summary Report
- Consolidate Multiple Branch Reports
- Export Final Report
